View and View Components

The following terminology is in use in the Ajs Framework:
Ajs Component HTML Synonym Description
View HTML Page Complete content of the HTML page, including the markup and style sheets
Visual Style Cascading Style Sheet Visual configuration of the information shown
Visual Component HTML component Ajs implementation of the HTML components similar to W3C component model
View Component JavaScript code ViewModel, also can be called as a code behind of the Visual Component
